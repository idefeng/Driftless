Driftless 隐私政策

生效日期：2026年6月30日
应用名称：Driftless
Android 包名：com.idefeng.driftless
版本：1.0.1

Driftless（以下简称“本应用”或“我们”）重视用户个人信息和隐私保护。 本政策说明我们在提供跑步步频节拍、训练计划、音色选择和播放控制等功能时， 如何处理与保护用户信息。

一、开发者与联系方式

运营主体：idefeng
联系邮箱：changdefeng06@gmail.com
联系地址：无

如您对本隐私政策或个人信息保护相关事项有疑问、意见、投诉或请求， 可通过上述联系方式与我们联系。我们将在收到请求并验证必要信息后， 于合理期限内处理并回复。

二、我们如何收集和使用信息

当前版本的 Driftless 不要求用户注册或登录账号，不收集姓名、手机号、 身份证号、邮箱、通讯录、精确定位、照片、麦克风、摄像头、生物识别信息、 医疗健康档案、支付信息或跑步轨迹。

1. 本地训练设置

为提供步频节拍和训练计划功能，本应用会在您的设备本地保存以下设置：

  • 目标步频、当前步频和步频调节偏好；
  • 训练阶段名称、阶段时长和阶段步频；
  • 节拍音色、播放模式、共存播放设置；
  • 应用语言、界面状态等基础偏好设置。

上述数据仅用于在本机恢复您的训练体验，保存在您的设备本地。 当前版本不将这些数据上传至服务器，也不用于广告、画像或定向推送。

2. 联系我们时提供的信息

如果您通过邮件或其他方式主动联系我们，我们可能会处理您提供的邮箱地址、 联系内容、问题描述、设备型号、系统版本、应用版本等信息。 这些信息仅用于回复咨询、定位问题、处理投诉或改进服务。

3. 第三方 SDK 与统计分析

当前版本未接入广告 SDK、统计分析 SDK、社交登录 SDK、支付 SDK、地图定位 SDK、 第三方推送运营 SDK 或第三方崩溃上报 SDK。我们不会向第三方出售、出租或共享 用户个人信息。

三、系统权限说明

为实现应用功能或满足 Android 系统运行要求，本应用可能声明或申请以下权限。 我们会遵循最小必要原则，仅在相关功能需要时使用权限。

权限名称 使用目的 是否会收集个人信息
网络访问权限（INTERNET） 来自基础应用框架和系统运行能力。当前版本不通过网络上传训练设置或个人信息。 当前版本不主动收集或上传个人信息。
通知权限（POST_NOTIFICATIONS） 用于在训练运行时显示持续通知、播放状态和快捷控制按钮，便于您在后台或锁屏状态下控制节拍。 不会读取通知内容，也不会收集其他应用通知信息。
前台服务权限（FOREGROUND_SERVICE / FOREGROUND_SERVICE_MEDIA_PLAYBACK） 用于在切换应用、息屏或后台运行时继续播放步频节拍，并保持训练控制通知。 不用于采集个人身份信息或位置轨迹。
震动权限（VIBRATE） 用于必要的系统交互反馈或训练控制反馈。 不收集个人信息。
读取/写入外部存储权限（READ_EXTERNAL_STORAGE / WRITE_EXTERNAL_STORAGE，Android 12 及以下） 该权限由基础构建框架声明。当前版本不会主动读取、扫描或上传您的照片、音频、视频、文件或媒体库内容。 当前版本不通过该权限主动收集个人信息。
悬浮窗权限（SYSTEM_ALERT_WINDOW） 该权限由 Android 构建配置声明。当前版本不会通过悬浮窗展示广告，也不会通过该权限采集个人信息。 如系统提示授权，您可以拒绝，不影响核心步频节拍功能。 不通过该权限收集个人信息。

四、我们如何存储信息

当前版本的训练设置和偏好数据保存在您的设备本地。我们不运营用于同步训练设置的云服务器， 不会将本地训练设置上传至远程服务器。

如果您卸载本应用，设备本地保存的应用数据通常会随系统卸载流程删除。 如果您通过邮件联系我们，相关邮件和沟通记录可能会在处理咨询、投诉或争议所需期间内保存。

五、我们如何共享、转让和公开披露信息

当前版本不会向第三方出售、出租、共享或转让您的个人信息，也不会公开披露您的个人信息。 如未来因法律法规、行政司法机关要求或业务功能变化确需处理个人信息， 我们会依法履行告知、取得同意或其他必要程序，并更新本政策。

六、您的权利

在适用法律法规允许的范围内，您可以行使以下权利：

  • 查询、复制、更正或删除您主动向我们提供的个人信息；
  • 撤回对非必要权限的授权；
  • 要求解释本政策中的个人信息处理规则；
  • 通过卸载应用删除本地保存的训练设置和偏好数据；
  • 通过本政策列明的联系方式向我们投诉或提出个人信息保护请求。

您也可以通过 Android 系统设置管理通知、存储、悬浮窗等权限。 拒绝非必要权限不会影响本应用的核心步频节拍功能，但可能影响后台通知控制等相关体验。

七、未成年人保护

本应用面向普通跑步训练和节奏辅助场景，不专门面向儿童提供服务。 如未成年人使用本应用，应在监护人指导下使用。当前版本不会主动收集未成年人的身份信息、 学籍信息、监护人信息或其他敏感个人信息。

八、政策更新

如后续版本新增账号登录、云同步、统计分析、广告、支付、地图定位、社交分享、 第三方 SDK 或其他涉及个人信息处理的功能，我们会更新本隐私政策， 并在应用内、应用商店页面或其他适当位置提示您阅读。

未经您同意，我们不会削减您按照本政策应享有的权利。 对于重大变更，我们会以显著方式进行提示。

九、适用法律与争议处理

本政策的订立、执行和解释适用中华人民共和国相关法律法规。 因本政策或个人信息保护事项产生争议的，您可以先通过本政策列明的联系方式与我们沟通解决。

十、特别提示

本政策根据当前版本功能、APK 权限声明和上架审核材料编写。 如应用实际功能、权限、第三方 SDK 或数据处理方式发生变化， 应在发布前同步更新本政策，确保隐私政策与应用实际行为一致。
