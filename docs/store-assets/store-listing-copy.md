# Driftless 上架文案

## 中文商店信息

应用名称：Driftless

副标题：稳定步频，训练不漂移

一句话简介：
Driftless 是一款为跑步者设计的极简步频节拍器，帮助你用稳定节奏完成训练。

短简介：
跑步步频节拍器，支持训练计划、音色选择和共存播放，帮你稳定节奏。

详细介绍：
Driftless 专注于一件事：让跑步节奏更稳定。

你可以设置目标步频，用清晰的节拍声跟随训练；也可以创建分阶段训练计划，在热身、巡航、冲刺之间自动切换步频。应用支持多种节拍音色，并提供共存播放模式，适合边听音乐边进行步频训练。

核心功能：
- 实时步频节拍，支持快速加减步频。
- 分阶段训练计划，适合热身、巡航和冲刺训练。
- 多种节拍音色，便于在不同环境中辨识节奏。
- 共存/独占播放设置，适配音乐播放场景。
- 简洁界面，无账号流程，打开即可训练。

适合人群：
- 想稳定跑步步频的跑者。
- 进行节奏跑、间歇跑、马拉松训练的人。
- 希望用声音提示替代频繁看表的人。

关键词：
跑步,步频,节拍器,跑步训练,节奏跑,间歇跑,马拉松,步频训练,运动健康

更新说明：
首个版本发布：支持步频节拍、训练计划、音色选择和共存播放设置。
